Chaque élément que vous voyez dans vos ***{{ fr.VLT }}s*** et ***{{ fr.PVLT }}*** est une entrée. Il existe plusieurs [Type d'entrées](/fr/server/web-interface/vault/entries/entry-type/) qui peuvent être créées directement dans l'interface web de {{ fr.DVLS }} .  

Gérer, importer, organiser et enregistrer toutes vos données avec des ***Entrées*** et des ***Dossiers*** dans votre {{ fr.DVLS }} .
