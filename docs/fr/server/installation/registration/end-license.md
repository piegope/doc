Certains de nos utilisateurs de {{ fr.DVLS }} pourraient se demander ce qu'il adviendra de leurs données lorsque la licence expirera.  

### Abonnement 

Tous les plans d'abonnement de {{ fr.DVLS }} offrent une maintenance du logiciel pendant la durée de l'abonnement. Ceci comprend toutes les mises à niveau majeures/mineures et l'assistance technique.  

Vous devez renouveler votre contrat d'abonnement à l'expiration. Si vous ne renouvelez pas votre abonnement, l'accès à vos données sera limité à lecture seulement.  

Cela dit, vous pourrez exporter vos données si vous choisissez de migrer vers un autre type de source de données que {{ fr.DVLS }}.
